## Install the dependencies

```bash
yarn
# or

```

### Start the app in development mode (hot-code reloading, error reporting, etc.)

```bash
quasar dev
# or
npx quasar dev
```

### Lint the files

```bash
yarn lint
# or
npm run lint
```

### Format the files

```bash
yarn format
# or
npm run format
```

### Build the app in PWA mode for production

```bash
quasar build -m pwa
```

### Start the app in PWA mode

```bash
quasar serve dist/pwa
```
